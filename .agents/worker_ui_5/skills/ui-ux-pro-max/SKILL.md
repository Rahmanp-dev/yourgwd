# UI/UX Pro Max Skill Copy

(This is a local reference copy for worker_ui_5)

Minimalist Luxury Style Guidelines:
- Abundant whitespace (margins of 8% or more on desktop, 16px-24px on mobile) to let the elements breathe.
- Clean thin border lines (typically `border-[1px] border-slate-200` or `border-neutral-800` depending on theme, or subtle lines separating content without blocky containers).
- Elegant muted tones (warm gray, beige, linen, slate, charcoal, cream, pale gold/bronze accents).
- Large hero text with high-quality serif or clean geometric sans-serif fonts.
- Sleek minimalist forms: border-b or thin outlined inputs with floating labels or very clean layout.
- High-quality SVG icons instead of emojis or decorative clutter.
- Responsive, accessible spacing based on a clear vertical rhythm.
