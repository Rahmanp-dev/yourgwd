# UI/UX Pro Max - Design Intelligence Local Copy

This is a copy of the design instructions for reference in this worker environment.

## Key Design Principles
1. **Accessibility**: Minimum 4.5:1 text contrast. Clear focus rings. Descriptive labels for icons.
2. **Touch & Interaction**: Target size >=44x44px, spacing >=8px. Async buttons show loader/disabled state.
3. **Typography**: Line height 1.5-1.75. Clear hierarchy.
4. **Style Selection**: Match product type.
   - Midnight Tech: Jet black, steel silver, subtle neon white/ice-blue glows.
   - Brutalist: Stark grids, high contrast primary colors, thick borders, offset shadows.
5. **No Emojis as Icons**: Use Lucide icons instead of emojis.
6. **Form UX**: Clear inline errors, submit button state transition (idle -> loading -> success).
