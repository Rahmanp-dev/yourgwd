## Core Rules & Checklist for Web UI
1. **Accessibility**:
   - Contrast ratio 4.5:1 for normal text.
   - Visible focus rings.
   - Alt text for images.
   - Keyboard nav support.
   - aria-label for icon-only buttons.
2. **Touch & Interaction**:
   - Target size at least 44x44px.
   - Spacing between interactive elements.
   - Spinner/feedback on async load (form submission).
   - Cursor pointer on clickables.
3. **Style Selection**:
   - Align shadows, border radius, colors with design goals.
   - SVG icons only (Lucide), no emojis as icons.
4. **Layout**:
   - Responsive breakpoints.
   - 8px spacing grid.
   - Avoid horizontal scroll on mobile.
5. **Typography & Color**:
   - Base size >= 16px.
   - Line height 1.5-1.75.
   - Primary/Secondary/Accent token usage.
6. **Forms**:
   - Labels visible.
   - Validation on blur, error messages below inputs.
   - Submit loading & success states.
