# UI/UX Pro Max - Design Guidelines

Refer to d:\GWD\Sales Machine\.agents\skills\ui-ux-pro-max\SKILL.md for the full contents.

Key highlights for this task:
- Strict light, warm, or pastel tones. Terracotta (#C2593F), Teal (#0D9488), Sand (#F5EBE6).
- Accessible text contrast: >= 4.5:1.
- Tap feedback on all buttons: subtle scale (0.95) or opacity transitions.
- Touch target minimum: 44x44px.
- Use Lucide icons, no emojis.
- Mobile-first responsive layout.
- Clear state validations on the Booking Form (required, email, phone, submit states).
