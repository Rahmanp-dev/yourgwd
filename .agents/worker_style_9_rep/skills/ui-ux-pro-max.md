# UI/UX Pro Max Skill Reference
This file is a local reference for the `ui-ux-pro-max` skill.
Key guidelines:
- Minimum touch target size: 44x44px, 8px spacing, active visual press states.
- SVG icons only (Lucide React), no emojis.
- Pair Display font with Body font.
- Responsive mobile-first layout.
- Validation, states, loading, feedback.
