# UI/UX Pro Max - Design Intelligence

Comprehensive design guide for web and mobile applications.
Priority checklist:
1. Accessibility: Contrast 4.5:1, Focus states, Alt text, Aria labels.
2. Touch & Interaction: targets >= 44x44px.
3. Performance: WebP, lazy loading, avoid layout shift.
4. Style Selection: Match product type. Electric Green/Aqua on Teal vs High-contrast Geometric Red/Dark Gray.
5. Layout & Responsive: Mobile-first.
6. Typography & Color: Clear pairings, readable line height.
7. Animation: 150-300ms transitions.
8. Forms & Feedback: Validation, error messages.
9. Navigation Patterns: Consistent nav, clear CTA.
10. Charts & Data: Clear viz.
