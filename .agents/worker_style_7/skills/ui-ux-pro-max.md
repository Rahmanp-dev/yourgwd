# UI/UX Pro Max Methodology (Local Copy)
Source: d:\GWD\Sales Machine\.agents\skills\ui-ux-pro-max\SKILL.md

## Key Checks:
1. Accessibility: Contrast 4.5:1, Alt text, Keyboard nav, Aria-labels. Avoid removing focus rings, icon-only buttons without labels.
2. Touch & Interaction: Min size 44x44px, 8px+ spacing, Loading feedback. Ripple/highlight press feedback.
3. Style Selection: Match product type, consistency, SVG icons (no emoji).
4. Layout & Responsive: Mobile-first breakpoints, Viewport meta, No horizontal scroll.
5. Typography & Color: Base 16px, Line-height 1.5, Semantic color tokens.
6. Animation: Duration 150-300ms, Motion conveys meaning, scale feedback (0.95-1.05) on press, exit faster than enter.
7. Forms & Feedback: Visible labels, Error near field, Helper text, Progressive disclosure.
8. Navigation Patterns: Predictable back, bottom nav, state-preservation.
9. Icons: SVG icons (Lucide), consistent icon sizing (stroke, size), alignment.
