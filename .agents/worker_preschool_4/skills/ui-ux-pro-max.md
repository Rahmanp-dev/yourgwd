# UI/UX Pro Max - Design Intelligence

Comprehensive design guide for web and mobile applications. Contains 50+ styles, 161 color palettes, 57 font pairings, 161 product types with reasoning rules, 99 UX guidelines, and 25 chart types across 10 technology stacks.

## Key Methodology Summary

1. **Accessibility (CRITICAL)**: Minimum 4.5:1 contrast ratio, visible focus rings, descriptive alt text, aria-labels for icon-only buttons, keyboard navigation.
2. **Touch & Interaction (CRITICAL)**: Minimum 44x44pt tap target size, 8px+ spacing, loading feedback on buttons, standard gestures.
3. **Performance (HIGH)**: Optimize images, declare dimensions/aspect-ratios, lazy load below the fold, progressive loading/skeleton screens.
4. **Style Selection (HIGH)**: Match style to product type. Avoid emojis as structural icons (use SVG/Lucide). Maintain icon style and stroke consistency.
5. **Layout & Responsive (HIGH)**: Mobile-first responsive layout, no horizontal scroll, consistent max-width on desktop, 8px spacing scale.
6. **Typography & Color (MEDIUM)**: Line-height 1.5-1.75, type scale, readable font pairings, semantic color tokens.
7. **Animation (MEDIUM)**: Micro-interaction duration 150-300ms, use transform/opacity only, exit faster than enter, responsive to reduced-motion.
8. **Forms & Feedback (MEDIUM)**: Input labels visible (no placeholder-only), error placement below field, submit loading feedback, inline validation on blur.
9. **Navigation Patterns (HIGH)**: Predictable back behavior, deep linking, active page highlighted, clear escape routes in modals.
10. **Charts & Data (LOW)**: Choose appropriate chart type, legends, tooltips, accessible contrast.
